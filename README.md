# Sway Focus Premium Website V2

Static website package for swayfocus.com.

## Deploy on GitHub Pages

1. Upload all files to your repository root.
2. Go to Settings → Pages.
3. Select Deploy from branch → main → root.
4. Make sure the custom domain is swayfocus.com.
5. Enable HTTPS after DNS is ready.

## Files

- index.html
- styles.css
- privacy.html
- support.html
- terms.html
- CNAME
- robots.txt
- sitemap.xml
- assets/
